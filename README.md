# mbox
> mbox 是一个基于mqtt协议的openwrt路由器管理客户端，使用libmosquitto C库实现  
> 请大家不吝关注

## 支持的功能
* 查看路由器信息：包含硬件型号、硬件版本、软件版本、1分钟/5分钟/15分钟CPU使用率、内存使用率、启动时间等
* 查看网络信息：包含外网联网方式、IP地址信息、内网IP地址等
* 查看WIFI信息：包含2.4G和5G的wifi详情，SSID、密码、加密方式等
* 查看下挂设备列表：下挂设备的mac、ip等信息
* 设置wan链接
* 设置lan地址和子网掩码
* 设置WIFI信息
* 重启设备
* 升级固件
* 升级自身程序
